CEZESSENTIALS COMMANDS

GENERAL
/cezhelp
Description: Show available commands
Permission: cezessentials.help

/cezreload
Description: Reload config and messages
Permission: cezessentials.reload


SPAWN
/spawn
Description: Teleport to spawn
Permission: cezessentials.spawn

/setspawn
Description: Set the server spawn
Permission: cezessentials.setspawn


HOMES
/home
Description: Teleport to your home
Permission: cezessentials.home

/sethome
Description: Set your home
Permission: cezessentials.sethome

/delhome
Description: Delete your home
Permission: cezessentials.delhome


WARPS
/warp <name>
Description: Teleport to a warp
Permission: cezessentials.warp

/setwarp <name>
Description: Set a warp
Permission: cezessentials.setwarp

/delwarp <name>
Description: Delete a warp
Permission: cezessentials.delwarp


TELEPORT REQUESTS
/tpa <player>
Description: Request teleport to a player
Permission: cezessentials.tpa

/tpaccept
Description: Accept teleport request
Permission: cezessentials.tpaccept

/tpdeny
Description: Deny teleport request
Permission: cezessentials.tpdeny
